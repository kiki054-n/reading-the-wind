## 1. TTT式とは何か

```
P = xX + yY + zZ + Uu + vV + wW
```

TTT式は、ある地点の「状態値 P」を、六つの独立したベクトルの和として記述する式である。

### 式の読み方

**P（状態値）**とは、その地点・その時点における「系の状態」を表す単一の値である。気象であれば大気の総合的な状態、組織であれば組織の総合的な健全度、社会であれば社会の総合的な安定度に相当する。

**各項（xX、yY…）**は、Pを構成する独立した次元の力・流れ・状態である。小文字（x, y, z, u, v, w）は係数（どの程度影響しているか）、大文字（X, Y, Z, U, V, W）はベクトルの方向・種類を表す。

### なぜ加算か

現実の系は、複数の力が同時に作用している。TTT式はそれらを**独立した成分に分解**することで、「どの力が支配的か」「どの力が欠落しているか」を診断しやすくする。

加算で記述することの意味は、各成分が**独立している**という仮定にある。ただしこの独立性は近似であり、成分間の相互作用は E式の `jJ`（交差項）で補完される。

### 静的記述の意味

TTT式は「ある瞬間の状態のスナップショット」を記述する。時間の流れを内包しない。

時間的な変化・動的な過程を記述するのは [E式](03_E_dynamic.md) の役割である。TTT式とE式は、静止画と動画の関係にある。

```
TTT式（静的）：状態の「今」を診断する
E式（動的）  ：状態が「どう変化しているか」を追う
```

## 2. 六つの変数の詳細

### xX — 水平方向の力・対立軸

**基本的な意味：** ある系の中で、横方向に働く力・勾配・対立の構造。

気象では東西方向の風・気圧勾配がこれに相当する。西から東へ流れる偏西風、気圧の高低差による水平方向の気流がxXである。

社会・組織では、同じ階層に属する勢力間の対立・競合・連携がxXである。与野党の対立、部門間の競合、地域間の経済格差などがこれに相当する。

**診断の問い：**
- 現在、水平方向のどの対立軸が最も張力を持っているか？
- その勾配（差）は拡大しているか、縮小しているか？
- xXが極端に大きい場合、衝突・統合のどちらが起きやすいか？

---

### yY — 縦方向の力・権威構造

**基本的な意味：** 上下方向に働く力・温度勾配・ヒエラルキーの構造。

気象では南北方向の温度勾配・前線の構造がyYである。寒気と暖気の境界（前線）は、温度という縦方向の勾配が横方向に走る境界線として現れる。

社会・組織では、権威・権力・情報の上下方向の流れがyYである。国家と市民、経営者と従業員、中央と地方の間に働く力がこれに相当する。

**日本の国家構造におけるyY：**
行政権（yY）の係数が立法（xX）・司法（zZ）を圧倒している状態が、現在の日本の診断結果である。法案の約8割が内閣提出であり、最高裁判事は内閣が任命する。yYが他の変数を吸収している非均衡状態。

**診断の問い：**
- 上から下への一方的な力か、双方向の関係か？
- yYが過大になっている場合、どの層に矛盾が蓄積しているか？

---

### zZ — 鉛直方向の力・チェック機能

**基本的な意味：** 垂直方向に働く力・上昇・下降・変換のメカニズム。

気象では鉛直方向の気流がzZである。地表の熱せられた空気が上昇する上昇気流、前線での強制上昇、積乱雲の発達がzZの正の方向への作用である。これが降水の主要なメカニズムとなる。

社会・組織では、ある層から別の層へ作用を「変換・増幅・抑制」するメカニズムがzZである。司法による行政のチェック、監査機能、内部告発制度、メディアの批判機能がこれに相当する。

**zZが機能しない場合：**
球体のある面が硬直し、押しても変形しない状態になる。日本の最高裁がその典型例である。戦後77年間で法令違憲判決13件という事実は、`zZ ≒ 0` の数字的証拠である。

zZがゼロに近づくと、yY（行政）への作用が裏面に伝達されなくなり、yYの一人勝ちの非対称球体になる。

**診断の問い：**
- この系のzZ（変換・チェック機能）は実際に機能しているか？
- 機能していない場合、それはzZの設計の問題か、運用の問題か、人事の問題か？

---

### Uu — 時間変化率・変化の加速度

**基本的な意味：** 状態が時間に対してどの速度で変化しているかを表す。

気象では気圧・気温の時間変化率がUuである。「気圧が急落している」「気温が急上昇している」という変化の速度そのものが、次の現象の予測に使われる。静的な気圧値より、気圧の変化速度の方が嵐の予測に有効なことがある。

社会・組織では、変化の速度・加速度がUuである。Uuがゼロに近い状態は「硬直・停滞」を意味する。Uuが急上昇している状態は「変革期・臨界への接近」を示す。

**Uuがゼロの危険性：**
日本の憲法は77年間改正されていない（Uu ≒ 0）。iI（潜在的な矛盾・蓄積）がいくら増大しても、Uuがゼロでは転換が起きない。しかし蓄積は止まらない。外部ショック（eE）がUuを強制的に上昇させた時に、急激な転換が起きる——これが55年体制以降の日本の構造的リスクである。

**診断の問い：**
- 現在のUuは正か、ゼロか、負（後退）か？
- Uuが急変したシグナルは何か？
- Uuをゼロに固定しているのは何か（慣性・制度・人事・文化）？

---

### vV — 流体・意思・フラックス

**基本的な意味：** ある次元を流れる「量」の方向と強さ。流体・意思・エネルギーの輸送。

気象では水蒸気フラックスがvVの典型例である。南西から暖湿気が大量に流入する時、vVの係数が増大し、降水の可能性が高まる。水蒸気は見えないが、露点・湿度計でその量を推定できる。

社会・組織では国民意思・市場心理・集合的な動機がvVである。個々の意見は見えにくいが、選挙結果・世論調査・消費行動・離職率などからその総量と方向を推定できる。

**vVとiIの関係：**
vVが十分に蓄積し（見えない圧力が高まり）、それが言語化・組織化される前の段階は、E式のiI（潜在量）に相当する。vVがiIからrRへ転換する瞬間が、世論の可視化・革命・選挙の地殻変動として現れる。

**診断の問い：**
- 現在、vVはどの方向に・どの速度で流れているか？
- その流れは制度（xX・yY・zZ）によって誘導されているか、それとも自発的か？
- vVが制度に抗する方向に流れている場合、どこで摩擦が生じているか？

---

### wW — エネルギー移流・正統性

**基本的な意味：** 系のエネルギー・動機・正統性の分布と移動方向。

気象では熱移流・潜熱がwWである。暖気の移流は気温を上昇させ、気象系のエネルギーを変化させる。台風が陸上に上陸して急速に衰弱するのは、エネルギー源（海面水温）からのwWが断たれるためである。

社会・組織では「正統性」「信頼」「動機」がwWである。ある権力が正統性を持つのは、wWが集中しているからである。正統性を失った権力は、wWを失った台風のように急速に衰弱する。

**wWの転換：**
令和の国体論が指摘する「国のために国民がいる」から「国民のために国家がある」への転換は、wW（正統性エネルギー）の流れの方向を逆転させることである。E式で言えば、ω（回転）の方向を反転させることに相当する。

**診断の問い：**
- 現在、wWはどこに集中しているか？
- wWの源泉（エネルギーを供給しているもの）は何か？
- wWが枯渇するとしたら、何がそれを起こすか？

## 3. 値ではなく有無を判断する

TTT式の最も重要な実践的原則：

> **正確な値を算出することより、各変数の「有無」を判断することが先である。**

気象数値予報モデルは、各変数を精密に計算しようとする。しかしその精密さが、「変数の存在自体を見落とす」という盲点を生む場合がある。

### 有無判断の手順

```
ステップ1：xX〜wWのそれぞれについて、この系に「存在するか」を問う
ステップ2：存在する場合、「正方向」か「負方向」か「ゼロ」かを判断する
ステップ3：「支配的な変数」はどれかを特定する
ステップ4：説明できない残差がある場合、第7項以降の変数の存在を疑う
```

### 存在判断のチェックリスト

| 変数 | 存在確認の問い | 判定 |
|------|-------------|------|
| xX | 水平方向の対立・勾配はあるか？ | 有 / 無 / 不明 |
| yY | 上下方向の権威・温度差はあるか？ | 有 / 無 / 不明 |
| zZ | 変換・チェック機能は実際に働いているか？ | 有 / 無 / 不明 |
| Uu | 状態は変化中か、停滞しているか？ | 有 / 無 / 不明 |
| vV | 意思・流体・フラックスの流れはあるか？ | 有 / 無 / 不明 |
| wW | エネルギー・正統性はどこに集中しているか？ | 有 / 無 / 不明 |

## 4. 三列比較による検証法

TTT式の予想と既存モデルの予想と実際の現象を、三列で並べて比較する。

```
┌─────────────┬──────────────────────┬──────────────────────┬──────────────────┐
│ 観測地点    │ TTT理論予報          │ データ予報           │ 実際の現象       │
│             │ （支配変数の特定）   │ （既存モデル・常識） │ （観測事実）     │
├─────────────┼──────────────────────┼──────────────────────┼──────────────────┤
│ 地点A       │ vV・zZ支配で予想①  │ 既存予報値           │ 実測値           │
│ 地点B       │ yY・hH支配で予想②  │ 既存予報値           │ 実測値           │
│ 地点C       │ Uu急変で予想③      │ 既存予報値           │ 実測値           │
└─────────────┴──────────────────────┴──────────────────────┴──────────────────┘
```

### 三列から読めること

**TTTとデータ予報が一致し、実際と乖離した場合：**
→ 両モデルに共通する盲点がある。見落とされた変数（第7項以降）が存在する可能性が高い。

**TTTのみが正解の場合：**
→ どの変数（どの係数の評価）がデータ予報と違ったかを特定する。その差が理論の優位点。

**データ予報のみが正解の場合：**
→ TTT式の変数の重み付けに改善余地がある。どの変数を過大・過小評価したか？

**両者とも外れた場合：**
→ 系に第7項以降の変数が存在する強い証拠。乖離の種類（量的・位置的・時間的・質的）から欠落変数を推定する。

## 5. 第7項以降の発見

TTT式の六変数で説明できない残差が繰り返し現れた時、第7項以降の変数の追加を検討する。

### 追加の判断基準（三条件）

1. **複数の独立した事例で繰り返し欠落している**
   一度だけの乖離は偶発的かもしれない。複数の事例で同じ種類の乖離が繰り返される場合、構造的な欠落を示す。

2. **有無だけで予測精度が変わる**
   正確な値は不明でも、「この変数が存在する/しない」の判断だけで、予想の方向性が大きく変わる。

3. **他の変数（xX〜wW）と独立している**
   既存の変数の組み合わせで表現できる場合は、新変数の追加ではなく係数の修正で対応する。

### 現在確認されている第7項以降の候補

| 項 | 変数名 | 発見した事例 | 意味 |
|----|-------|-----------|------|
| **hH** | 前歴・履歴 | 能登大雨（地盤脆弱化）/ 砂川事件（密談の前歴） | 系の過去状態が現在の応答パターンを変える |
| **sS** | スケール・規模 | 気象の局地/広域差 / 個人/組織/社会の違い | 現象の空間・時間スケールが結果を変える |
| **eE** | 境界条件・外部環境 | 占領下での憲法制定 / 地形による気象変化 | 系の外側から課される制約・強制入力 |
| **nN** | 非線形臨界 | 積乱雲の爆発的発達 / 革命・制度崩壊 | 閾値を超えると質的に変化する転換点 |
| **cC** | 連鎖・カスケード | 台風と前線の相互強化 / サプライチェーン連鎖 | 隣接する系との相互作用による増幅 |
| **dD** | 散逸・抵抗・減衰 | 摩擦・放射冷却 / 組織の疲弊・既得権抵抗 | エネルギーが失われる方向と速度 |
| **aA** | AI・技術基盤 | デジタル化による社会構造の変化 | 技術が基盤化することでyY・zZの一部を代替 |

→ 各変数の詳細な考察：[theory/04_variables.md](04_variables.md)

## 7. TTT式の限界とE式への接続

### TTT式の限界

**限界①：時間を扱えない**
TTT式はある瞬間の状態を記述する。変化の過程・履歴・将来の軌跡を記述するには不十分である。

**限界②：相互作用を扱えない**
各変数を独立と仮定している。実際には変数間に相互作用がある（気温上昇が水蒸気を増やし、水蒸気が熱を保持する、など）。

**限界③：潜在量を扱えない**
rR（顕在量）は記述できるが、iI（潜在量）を直接記述できない。「見えない裏面」そのものを式の中に取り込めない。

### E式への接続

これらの限界を補うのがE式である。

```
E = (xX + yY + zZ) ⊗ (ωx, ωy, ωz) + rR + iI + jJ
```

| TTT式の限界 | E式の補完 |
|-----------|---------|
| 時間を扱えない | ⊗テンソル積とωで「変化の方向と速度」を記述 |
| 相互作用を扱えない | jJ（交差項）で変数間の相互作用を記述 |
| 潜在量を扱えない | iI（虚部）で見えない蓄積量を記述 |

→ E式の詳細：[theory/03_E_dynamic.md](03_E_dynamic.md)

## 8. 実践：自分の領域にあてはめる

TTT式を自分の領域に適用する手順を示す。

### 手順1：P（状態値）を定義する

あなたの領域で「状態の総合値」として何を定義するか。
- 気象なら：大気の不安定度
- 組織なら：組織の健全度・持続可能性
- 社会なら：社会的安定度・変革の成熟度

Pは単一の値である必要はない。「Pが高い＝○○、Pが低い＝××」という対応を最初に決める。

### 手順2：六変数をあてはめる

xX〜wWのそれぞれについて、あなたの領域での意味を具体的に書く。

```
私の領域（　　　　）でのTTT式：

P = xX(　　　　　) 
  + yY(　　　　　) 
  + zZ(　　　　　) 
  + Uu(　　　　　) 
  + vV(　　　　　) 
  + wW(　　　　　)
```

### 手順3：支配的な変数を特定する

現在の状態で、最も大きな係数を持つ変数はどれか。その変数が支配的な状態で、Pはどのような値を取るか。

### 手順4：欠落している変数を探す

六変数で説明できない「残差」は何か。それは第7項以降の変数（hH・sS・eE・nN・cC・dD）のどれかで説明できるか。

### 手順5：三列比較で検証する

予想したP値と、既存の予測と、実際の結果を三列で記録する。乖離の種類を特定し、欠落変数の候補を絞る。

→ 記録には [log/template.md](../log/template.md) を使う。
