## 1. なぜ第7項以降が必要か

TTT式の六変数（xX〜wW）は、多くの事象を説明できる。しかし検証を重ねるうちに、六変数では説明できない「残差」が繰り返し現れた。

### 残差が現れた事例

**能登大雨（2024年9月）**
気象変数（xX〜wW）は正確に把握できていた。しかし被害規模が予想を大幅に超えた。気象以外の要因——地震後の地盤脆弱化——が見落とされていた。これが hH（前歴）発見の契機となった。

**日本の司法機能不全**
三権（xX・yY・zZ）の構造を分析した時、「なぜzZがこれほど機能しないか」は六変数だけでは説明しきれなかった。GHQによる制度設計（eE：外部環境）と、砂川事件以来の統治行為論の定着（hH：前歴）という二つの変数が欠落していた。

**社会OS転換の困難**
vV（国民意思）を高めても、Uu（変化率）がゼロに固定されていると転換が起きない。このUuを固定している力——制度的慣性・既得権の抵抗——が dD（散逸・抵抗）として明示的に扱われる必要があった。

### 第7項以降の位置づけ

```
P式（基本）：
P = xX + yY + zZ + Uu + vV + wW

P式（拡張）：
P = xX + yY + zZ + Uu + vV + wW
  + hH + sS + eE + nN + cC + dD + aA + ...
```

ただし、すべての変数を一度に扱うことは実践的でない。検証の中で「この事例に存在する変数」を特定し、有無を判断してから使う。これが「値より有無を先に判断する」原則の応用である。

## 2. 変数追加の判断基準

新しい変数を第7項以降に追加するかどうかは、次の三条件で判断する。

### 条件①：複数の独立した事例で繰り返し欠落している

一度だけの乖離は偶発的かもしれない。しかし異なる領域・異なる事象で、同じ種類の乖離が繰り返される場合、構造的な欠落を示している。

hHは気象（能登）・司法（砂川事件）・社会（戦後憲法体制）という三つの独立した領域で同様に欠落していた。これが「確定済み」とする根拠である。

### 条件②：有無だけで予測精度が変わる

正確な値は不明でも、「この変数が存在する/しない」の判断だけで、予想の方向性が大きく変わる。

能登事例で「hHあり（地震後の地盤脆弱化）」と判断するだけで、「同じ降水量でも被害が大きくなる」という予想の方向性が得られた。値がわからなくても、方向性を変えるだけで十分に有用である。

### 条件③：既存の変数（xX〜wW）と独立している

既存の変数の組み合わせや係数の修正で表現できる場合は、新変数の追加ではなく係数の調整で対応する。

eE（外部環境）は、xXやyYとは独立している。xX（水平方向の力）がシステム内部の力を記述するのに対し、eEはシステムの外から課される制約・境界条件である。この独立性が追加を正当化する。

### 追加の手順

```
Step 1：乖離を特定する
         どの種類の乖離か（量的・位置的・時間的・質的）

Step 2：既存変数で説明できるか試みる
         係数の修正・組み合わせで説明できる場合は変数追加不要

Step 3：複数事例での確認
         同じ種類の乖離が他の事例でも現れるか

Step 4：独立性の確認
         既存変数と本質的に異なる次元を記述しているか

Step 5：命名と定義
         変数の意味・気象での対応・社会での対応を明示する
```

## 3. 確定済み変数：hH（前歴・履歴）

**状態：確定（複数事例で検証済み）**

### 定義

系の過去の状態・変形の履歴が、現在の入力に対する応答パターンを変える変数。

```
P_応答 = P_入力 × f(hH)

f(hH) > 1 ：前歴により脆弱化（同じ入力でより大きな変形）
f(hH) = 1 ：前歴なし・通常状態
f(hH) < 1 ：前歴により強化（同じ入力でより小さな変形）
```

### 気象での意味

| hHの内容 | 影響 |
|---------|------|
| 地震後の地盤脆弱化 | 同じ降水量でより大きな土砂崩れが起きる |
| 前年の少雨による地下水位低下 | 農業被害の規模が変わる |
| 過去の洪水で堆積した土砂 | 河川の流量と氾濫危険水位が変わる |
| 長期的な地温上昇の蓄積 | 積乱雲の発達ポテンシャルが上昇している |

### 社会・組織での意味

| hHの内容 | 影響 |
|---------|------|
| 戦争・占領の前歴（日本の憲法体制） | 同じ制度的入力でも、前歴がない国より変革抵抗が大きい |
| 過去の失敗経験（組織） | 新しい提案への拒絶反応が強くなる |
| 長期にわたる不信感の蓄積 | 信頼回復のコストが通常の何倍もかかる |
| 地震後の地盤のアナロジー（社会） | 過去のショックで脆弱化した社会は、次のショックへの耐性が低い |

### hHが「見えない」理由

hHは過去に起きたことである。現在の観測では見えない。しかし現在の応答パターンを見ることで、「なぜこの規模の変形が起きたか」の説明変数として推定できる。

> **「なぜ予想より大きな変形が起きたか」という問いに対し、hHが答える。**

### 他の変数との相互作用

```
hH × vV → 国民意思の表出が、前歴（過去の弾圧・失敗）によって抑制される
hH × zZ → 司法の独立性が、過去の人事慣行（hH）によって制約される
hH × nN → 臨界値が、前歴によって変化する（脆弱化すれば臨界が下がる）
```

## 4. 確定済み変数：sS（スケール・規模）

**状態：確定（複数事例で検証済み）**

### 定義

現象が展開する空間的・時間的スケールが、結果の性質を変える変数。同じ種類の現象でも、スケールが異なれば支配的なメカニズムが変わる。

### スケールの階層

```
空間スケール：
  局地（数km）→ 都市（数十km）→ 地域（数百km）→ 広域（数千km）→ 全球

時間スケール：
  瞬間（秒）→ 短期（時間）→ 中期（日〜週）→ 長期（月〜年）→ 超長期（年代）
```

### スケールによるメカニズムの変化

**気象でのsS：**

| スケール | 支配的なメカニズム | TTT変数での対応 |
|---------|----------------|--------------|
| 局地（数km） | 地形効果・熱対流 | zZ（地形変換）が支配的 |
| 中規模（数十〜百km） | 前線・低気圧 | yY（南北勾配）が支配的 |
| 広域（数千km） | ジェット気流・気団 | xX（東西循環）が支配的 |
| 全球 | 大気大循環・季節 | wW（熱移流の全球分布）が支配的 |

**社会でのsS：**

| スケール | 支配的なメカニズム | 対応する問い |
|---------|----------------|----------|
| 個人 | 動機・心理 | vV（個人の意思）が支配的 |
| 組織 | 文化・規範・インセンティブ | yY・wW（構造とエネルギー）が支配的 |
| 地域社会 | 慣習・地縁・行政 | hH・eE（前歴と境界条件）が重要 |
| 国家 | 制度・法・権力 | xX・yY・zZ（三権）が基本構造 |
| 国際 | 条約・同盟・覇権 | eE（外部環境）が支配的 |

### sSが重要な理由

異なるスケールの現象を同じ式で扱おうとすると、変数の意味が変わってしまう。

局地的な集中豪雨（sS：局地）を、広域的な気圧配置（sS：広域）の変数で説明しようとすると、本質的に支配的な変数（zZ：地形効果）が見えなくなる。

「どのスケールで見るか」を最初に決めることが、TTT式の変数選択の前提条件である。

## 5. 確定済み変数：eE（境界条件・外部環境）

**状態：確定（複数事例で検証済み）**

### 定義

分析対象の系の**外側**から課される制約・強制入力・境界条件を記述する変数。系の内部変数（xX〜wW）とは独立して、系の応答を変える。

### eEとxX〜wWの違い

```
xX〜wW：系の内部で働く力・流れ・エネルギー
eE    ：系の外側から課される制約・境界条件

例：
  気象  → xX〜wWは大気の内部状態
          eEは地形・海面水温・大陸配置（変えられない外部条件）

  組織  → xX〜wWは組織内部の力学
          eEは市場・規制・競合他社（組織が単独では変えられない外部条件）

  国家  → xX〜wWは国内の政治・経済・社会
          eEは国際条約・同盟・地政学的制約
```

### eEの特殊なケース：強制的なOS書き換え

eEが極端に大きくなる場合——外部からの圧倒的な強制——は、系の内部構造（ωを含む）を一時的または永続的に書き換える。

**最も顕著な事例：1947年の日本国憲法制定**

GHQ（eE）が、日本の国内的な合意形成プロセス（vV）を経ずに、憲法という最上位の構造を設計した。これはE式で言えば、`eE`が`ω（系の回転構造）`を外部から強制的に書き換えたことに相当する。

```
eE（GHQ）⊗ ω_旧体制 → ω_戦後体制（hHとして定着）
```

この「外部強制によるω書き換え」の痕跡が、75年後の現在でも`hH`として制度の中に生き続けている。

### eEの正・負の方向

```
eE正（有利な外部条件）：
  → 国際的な追い風（技術・経済・安全保障）
  → 良好な地政学的位置
  → 外部からの支援・資源流入

eE負（不利な外部条件）：
  → 国際的な制裁・孤立
  → 外部からの強制・占領
  → 資源の枯渇・市場の喪失
```

## 6. 要検討変数：nN（非線形臨界）

**状態：要検討（概念は確立、定量化は困難）**

### 定義

系が**質的に変化する閾値**。nNを超えると、量的な変化ではなく質的な転換が起きる。E式の`iI→rR転換`が起きる条件値。

### nNの性質

```
iI の蓄積量 < nN ：表面に変化は現れない（iIとして潜在したまま）
iI の蓄積量 ≥ nN ：急激にrRへ転換（質的変化・非可逆的）
```

nNは系によって異なり、また系の状態（hH・dD）によっても変化する。

**nNが下がる条件（転換が起きやすくなる）：**
- hHによる脆弱化（過去の変形が内部構造を弱体化）
- dDの低下（散逸・抵抗が弱まる）
- cC（連鎖）による蓄積の加速

**nNが上がる条件（転換が起きにくくなる）：**
- hHによる強化（過去の成功が内部構造を強化）
- dDの増大（散逸・抵抗が強まる）
- 系の分断（iIが局所にとどまり全体に波及しない）

### nNが「要検討」である理由

nNの正確な値を事前に知ることは、原理的に困難である。「転換が起きた後」にしか、nNを超えていたことがわからない場合がほとんどである。

しかし「nNの概念を持つこと」と「nNを正確に計算できること」は別の問題である。nNを意識することで、「iIが蓄積している」という認識が生まれ、転換の準備ができる。これだけでも十分に実践的な価値がある。

### nNの推定方法

| 間接指標 | 内容 |
|---------|------|
| 過去の類似事例との比較 | 同種の転換が過去にどのような条件で起きたか |
| 複数変数の同期 | 複数のiIが同時に臨界に近づいているか |
| 外部観察者の評価 | 系の外部にいる観察者が「もうすぐ転換が起きる」と感じているか |
| モデルの精度低下 | 既存の予測モデルが突然当たらなくなる |

## 7. 要検討変数：cC（連鎖・カスケード）

**状態：要検討（特定事例で有効、普遍化は継続中）**

### 定義

隣接する系との相互作用により、一つの系の変化が別の系に波及・増幅されるメカニズム。

```
系A の変化 → 系B への伝播 → 系B の変化 → 系C への伝播 → ...
```

### cCが重要な場面

**気象でのcC：**
台風（系A）が前線（系B）を強化し、前線強化が局地的な積乱雲（系C）を爆発的に発達させる。単独では発達しなかった積乱雲が、連鎖によって発達する。能登大雨での「台風変化低気圧＋秋雨前線」の相互作用がこれに相当する。

**社会・組織でのcC：**
- サプライチェーン連鎖（部品不足→製造停止→販売不振→雇用削減）
- 感染症の連鎖感染（一人の感染者から指数的な拡大）
- 金融危機の連鎖（一行の破綻→信用収縮→連鎖倒産）
- 社会運動の連鎖（一地域の運動が隣接地域へ伝播）

### cCとjJの違い

E式の`jJ`（交差項）は、**同一の系内**での変数間相互作用を記述する。  
`cC`は、**異なる系の間**での連鎖・伝播を記述する。

この違いが重要である。cCは、分析対象の「境界をどこに引くか」という問題と密接に関係する。

## 8. 要検討変数：dD（散逸・抵抗・減衰）

**状態：要検討（概念は確立、特定が困難な場合がある）**

### 定義

系の中でエネルギー・変化の勢い・iIの蓄積が失われていく方向と速度。Uuを下げる方向に働く力の総体。

```
dD が大きい ：変化のエネルギーが速く失われる（転換が起きにくい）
dD が小さい ：変化のエネルギーが持続する（転換が起きやすい）
```

### dDの具体例

**気象でのdD：**
- 大気摩擦（風速を減衰させる）
- 放射冷却（夜間に熱を失う）
- 乾燥化（水蒸気vVを減らす）
- 地表の粗度（地形による摩擦）

**社会・組織でのdD：**

| dDの内容 | 効果 |
|---------|------|
| 既得権益の抵抗 | 変革エネルギー（wW）を消耗させる |
| 官僚的慣性 | Uu（変化率）を常にゼロに引き戻す力 |
| 情報の非対称性 | vV（国民意思）が制度に届かない |
| 疲弊・無関心 | iI（変革への欲求）が蓄積されずに自然減衰する |
| 分断・孤立化 | cC（連鎖）が起きる前にエネルギーが拡散する |

### dDが「見えにくい」理由

dDは「何かが起きている」のではなく「何かが起きない」ことで存在を示す。変革が起きないのは、vVが弱いからか、dDが強いからか——この区別が難しい。

しかし区別することで対処法が変わる。
- vVが弱い → 意識啓発・ビジョンの共有が有効
- dDが強い → 抵抗勢力の特定と無力化が先決

## 9. 候補変数：aA（AI・技術基盤）

**状態：候補（現代的な新変数として検討中）**

### 定義

AI・デジタル技術が社会の基盤インフラとして機能する程度と、その影響方向を記述する変数。技術が基盤化することで、yY・zZの一部を代替または変換する。

### aAが独立変数として必要な理由

従来の技術革新は、xX〜wWのいずれかの変数の係数を変えるものとして扱えた（例：印刷技術→vVを増幅、鉄道→sSを変化）。

しかしAIは単一の変数の係数変化ではなく、**複数の変数の関係構造そのもの**を変える可能性がある。

```
aA が大きい場合の変化：
  yY（行政・官僚）の一部をAIが代替 → yYの実質的な係数変化
  zZ（司法・チェック機能）にAI審査が加わる → zZの経路多様化
  vV（国民意思）のAIによる集約・分析 → vVの可視化と増幅
  Uu（変化率）がAIによって加速 → Uuの構造的な上昇
```

### aAと令和の国体論

令和の国体論における「防衛技術の民間開放（風の共有）」は、aAの具体的な適用である。

AI・ICT技術が社会基盤として普及することで、これまでyY（行政・国家）が独占していた情報処理・意思決定の一部を、vV（市民・民間）が担えるようになる。これはyYの係数低下とvVの係数上昇を同時にもたらす。

```
aA導入前：P = ... + 大係数yY + 小係数vV + ...
aA導入後：P = ... + 中係数yY + 中〜大係数vV + aA_基盤 + ...
```

### aAの課題

- AIが特定の主体（国家・大企業）に集中した場合、yYをさらに肥大化させる可能性
- aAの係数が急速に変化するため、P式の静的記述に馴染みにくい
- E式のω（回転構造）にaAがどう影響するかの理論化が必要

## 10. 変数間の相互作用マップ

各変数がどう影響し合うかを示す。矢印の方向は「影響を与える方向」、「＋」は増幅、「−」は抑制。

```
【基本六変数の相互作用】

xX ──→ yY（水平の力が縦方向の構造を変える）
yY ──→ zZ（権威構造がチェック機能を規定する）
zZ ──→ xX（チェック機能が権力の対立軸を調整する）

vV ──→ Uu（意思・流体の蓄積が変化を駆動する）
Uu ──→ wW（変化速度がエネルギーの再配分を起こす）
wW ──→ vV（エネルギー・正統性が意思の方向を変える）

【第7項以降との相互作用】

hH ──→（全変数）の応答関数を変える
eE ──→ ω（系の回転構造を外部から書き換える可能性）
nN ──（iI蓄積量）との比較で転換を決定する閾値
cC ──→ 系境界を越えてxX〜wWを隣接系に伝播する
dD ──−→ Uu（変化率を常にゼロ方向に引く）
aA ──→ yZ比率（yYとvVのバランスを変える）

【特に重要な相互作用ペア】

hH × nN：前歴が脆弱化するほど、臨界（nN）が下がる
vV × eE：国民意思と外部ショックが重なると変革が加速する（jJ）
dD × Uu：散逸が大きいほど変化率が維持されない
aA × vV：技術基盤が整うほど国民意思が制度に届きやすくなる
```

## 11. 未発見変数の探索法

現時点で確認されている変数（hH・sS・eE・nN・cC・dD・aA）は、これで完全ではない。未発見の変数はまだ存在する可能性が高い。

### 未発見変数が存在するシグナル

1. **質的乖離の繰り返し**
   量的・位置的・時間的ではなく、「現象の種類そのものが違う」という乖離が繰り返される時

2. **既存変数すべての調整でも説明できない残差**
   係数をどう調整しても、繰り返し同じ方向の残差が残る

3. **複数の独立した領域での同時的な乖離**
   気象・司法・社会という異なる領域で、同じパターンの乖離が現れる

4. **「説明できないが確かに感じる」直感**
   第一層（感知）で「何かがある」という直感が、第二層（推論）で言語化できない場合

### 新変数の命名規則

発見した変数は、次の形式で命名する。

```
形式：[小文字][大文字]（例：hH、sS、eE）

小文字：係数を表す文字（その変数がどの程度影響しているか）
大文字：ベクトルの種類を表す文字（その変数が何を記述しているか）

条件：
  - 既存の変数（x, y, z, u, v, w, h, s, e, n, c, d, a）と重複しない
  - 変数の意味を反映した文字を選ぶ
  - 気象・社会の両方に対応する意味を持つことが望ましい
```

### このリポジトリへの投稿

新しい変数の候補を発見した場合、[Issues](https://github.com/kiki054-n/reading-the-wind/issues) に投稿してほしい。

```
Issue の書き方：

タイトル：[新変数候補] 変数名と概要

本文：
- 発見した事例：
- 既存変数で説明できなかった残差：
- 提案する変数の定義：
- 気象での対応：
- 社会・組織での対応：
- 他の変数との独立性の根拠：
```

## 12. 拡張されたP式・E式

検証を通じて発見された変数を組み込んだ、現時点での拡張式を示す。

### 拡張P式

```
P = xX + yY + zZ + Uu + vV + wW        （基本六変数）
  + hH                                   （前歴・履歴：確定）
  + sS                                   （スケール・規模：確定）
  + eE                                   （境界条件・外部環境：確定）
  + nN                                   （非線形臨界：要検討）
  + cC                                   （連鎖・カスケード：要検討）
  + dD                                   （散逸・抵抗：要検討）
  + aA                                   （AI・技術基盤：候補）
  + ...                                  （未発見変数）
```

ただし実践では、すべての変数を使う必要はない。**事例に存在する変数のみを使う。**有無の判断が先、値の計算が後、未使用変数は省略する。

### 拡張E式

```
E = (xX + yY + zZ) ⊗ (ωx, ωy, ωz)
  + rR                    （顕在量）
  + iI                    （潜在量：hH・nN・dDの影響を受ける）
  + jJ                    （相互作用：特にvV×eE、hH×nN）
  + f(eE)                 （外部環境による境界条件の修正項）
  + g(hH)                 （前歴による応答関数の修正項）
```
