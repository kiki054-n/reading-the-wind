## はじめに — この理論が生まれた問い

**OOπの表面を押せば、見えない裏面の変形が予想される。**

これが、この理論の出発点である。

ある現象を観測した時、その現象の「表面」だけを見ていては、次に何が起きるかを予想できない。押した面とは反対側の「裏面」がどう変形しているかを読むことが、予想の本質である。

気象予報士は、気圧・気温・湿度という「表面」から、数時間後の雨という「裏面」を読む。
裁判官は、法律の文言という「表面」から、社会への影響という「裏面」を読まなければならない。
組織のリーダーは、数字という「表面」から、構成員の意欲・疲弊・離脱という「裏面」を読む。

**見えないものを読む能力は、特定の領域の専門知識ではなく、思考の構造の問題である。**

このリポジトリは、その思考構造を言語化・数式化・検証するための記録である。

## 理論の三つの柱

### 第一の柱：OOπ理論 — 見えない裏面を読む

OOπとは、観測可能な表面への作用から、観測できない裏面の変形を予想する思考法である。

```
表面への作用
    ↓
[見えない内部構造]
    ↓
裏面の変形（予想）
    ↓
実際の現象との照合
    ↓
乖離の分析 → 理論の更新
```

実践は三つの層からなる。

| 層 | 名称 | 内容 |
|----|------|------|
| 第一層 | **感知** | データより先に、異変のシグナルを身体で掴む |
| 第二層 | **推論** | 表面の変化から、裏面の変形パターンを論理的に構成する |
| 第三層 | **フライトプラン** | 複数の予想シナリオから、最も確率の高い経路を選ぶ |

→ 詳細：[theory/01_OOpi.md](theory/01_OOpi.md)

### 第二の柱：TTT式 — 空間と状態の静的記述

```
P = xX + yY + zZ + Uu + vV + wW
```

空間・状態を6次元ベクトルで記述し、観測地点のデータから関係する地点の値を推定する。

| 変数 | 基本的な意味 | 気象への対応 | 社会・組織への対応 |
|------|------------|------------|-----------------|
| **xX** | 水平方向の力 | 東西風・気圧勾配 | 権力の横方向の流れ・対立軸 |
| **yY** | 縦方向の力 | 南北温度勾配・前線 | 上下の権威構造・ヒエラルキー |
| **zZ** | 鉛直方向の力 | 上昇気流・下降流 | 制度的チェック機能・司法 |
| **Uu** | 時間変化率 | 気象の時間的傾向 | 変化の速度・加速度・勢い |
| **vV** | 流体・意思の流れ | 水蒸気フラックス | 国民意思・市場心理・動機 |
| **wW** | エネルギー移流 | 熱移流・潜熱 | 正統性・信頼・動機のエネルギー |

**第7項以降**（hH：前歴、sS：規模、eE：境界条件、nN：非線形臨界、cC：連鎖、dD：散逸…）は、予想と現実の乖離を検証することで発見されていく。

> 正確な値を算出する以上に、**各変数の「有無」を判断すること**が、この式の実践的な使い方である。

→ 詳細：[theory/02_TTT_static.md](theory/02_TTT_static.md)

### 第三の柱：E式 — 状態の動的記述

```
E = (xX + yY + zZ) ⊗ (ωx, ωy, ωz) + rR + iI + jJ
```

| 要素 | 意味 |
|------|------|
| **⊗ テンソル積** | 空間3成分が角速度ベクトルと結合。回転・渦・変化の方向を記述する |
| **rR** | 実部・観測可能な顕在量。「表面」に現れているもの |
| **iI** | 虚部・観測できない潜在量。「裏面」に蓄積しているもの |
| **jJ** | 交差項・複数の力の相互作用。単独では見えない結合効果 |

TTT式が「今の状態のスナップショット」であるとすれば、E式は「状態がどう変化しているかの軌跡」を記述する。

`iI`（潜在量）が閾値を超えて`rR`（顕在量）に転換する瞬間——これが予想の核心である。積乱雲の爆発的発達、組織の突然の崩壊、社会制度の転換点——すべてこの構造を持つ。

→ 詳細：[theory/03_E_dynamic.md](theory/03_E_dynamic.md)

## 検証の方法 — 三列比較

予想と現実を三列で並べ、乖離を記録する。

| 観測地点 | TTT理論予報（支配変数） | データ予報（既存モデル） | 実際の現象 |
|---------|---------------------|---------------------|---------|
| 地点A | 支配変数から導かれる予想 | 既存のモデル・常識的な予想 | 観測された事実 |
| 地点B | 〃 | 〃 | 〃 |

**乖離の四分類：**

| 種類 | 内容 | 示唆する問題 |
|------|------|------------|
| 量的乖離 | 大きさが違う | どの変数の係数が誤っているか |
| 位置的乖離 | 場所が違う | 空間解像度の問題か、別の変数の介在か |
| 時間的乖離 | タイミングが違う | Uu（変化率）の読み誤りか |
| 質的乖離 | 現象の種類が違う | 式にない変数（第7項以降）の存在を示す |

> **乖離の中に、次の変数が眠っている。**

## 検証済み領域と主な発見

### 🌧 気象予報（2024年 山形・能登大雨）

- `vV × zZ` の積が臨界値を超えた時点で線状降水帯の形成が予想できた
- 地形による強制上昇（yY→zZ変換）が山間部で過小評価されやすい「裏面の変形」
- 能登事例では、地震後の地盤脆弱化という **hH（前歴）という第7項** が欠落していたことが判明

→ [verification/weather_2024.md](verification/weather_2024.md)

---

### ⚖️ 司法機能不全（日本最高裁）

- 三権分立という「球体」を押した時、xX・zZはほとんど変形せず、yYだけが過大に変形する**非対称球体**になっている
- 戦後77年間で法令違憲判決13件という事実が、**zZ係数≒0** の数字的証拠
- 統治行為論は「高度の政治性」という名のもとにzZを自己無効化する装置

→ [verification/judiciary_japan.md](verification/judiciary_japan.md)

---

### 🏛 砂川事件・統治行為論（1959年）

- **eE（外部圧力）がzZを上書きした**構造的事件
- 田中最高裁長官と米側の密談が、iI（潜在的な外部介入）として存在し、判決というrRに転換した
- この一事件が66年間にわたりiI（潜在規範）として日本の司法構造を規定し続けている

→ [verification/sunagawa.md](verification/sunagawa.md)

---

### 🇩🇪 日独司法比較

- ドイツはhH（ナチズムの前歴）をzZの設計原理に統合した
- 日本はhHを回避することでzZを無効化した
- **同じ敗戦国でも、前歴への向き合い方がzZ係数の差として70年後に現れている**

→ [verification/germany_compare.md](verification/germany_compare.md)

## 風を読む — 実践的な七つの問い

あなた自身の領域で、次の問いを立ててみてほしい。

1. **今、表面として観測できているものは何か？**
2. **その表面を押した時、どの方向に・どの程度・裏面が変形するか？**
3. **支配的な変数（xX〜wW）はどれか？**
4. **見えていない変数（iI）はどこに潜んでいるか？**
5. **前歴（hH）は現在の状態に影響しているか？**
6. **変化率（Uu）は加速しているか、減速しているか、ゼロに近づいているか？**
7. **予想と現実が乖離した時、欠落していた変数は何か？**

## 記録の方法 — ログの書き方

[log/template.md](log/template.md) を使い、四段階で記録する。

```
① 予想前の記録
   観察した表面 → TTT/E式の適用 → フライトプランの設計

② 予想後の記録（現象が起きた後）
   実際に起きたこと → 予想との照合

③ 乖離の分析
   量的・位置的・時間的・質的のどの乖離か → 欠落変数の特定

④ 理論への反映
   追加すべき変数・修正すべき係数・次回の予想への教訓
```

**外れた記録の方が、当たった記録より理論を育てる。**
乖離こそが発見の入口である。
