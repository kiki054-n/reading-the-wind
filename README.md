## この理論の目的

**表面を押すと、見えない裏面がどう変形するかを予想すること。**

OOπの球体の表面に作用を加えると、見えない裏面に変形が生じる。その変形を予想するために、見えないものを「感じる・推論する・設計する」方法を考察・記録するリポジトリである。

天気予報が「観測できない大気の立体構造を点のデータから再構成する作業」であるように、この理論の目的は**予想**である。予想は証明できないが、予想と現実の乖離を記録し続けることで、理論は精度を増す。

---

## 三つの核心的概念

### OOπ理論
球体の表面を押した時、裏面の変形を予想する思考法。
- **見える表面**：観測できる事象・データ・現象
- **見えない裏面**：その結果として生じる変化・連鎖・転換
- **予想の設計**：表面への作用から裏面の変形パターンを描く

### TTT式（静的記述）
```
P = xX + yY + zZ + Uu + vV + wW
```
空間・状態を6次元ベクトルで記述する。観測地点のデータから、関係する地点の値を推定し、状態変化を予想する。

| 変数 | 基本的な意味 | 気象 | 社会・組織 |
|------|------------|------|-----------|
| xX | 水平方向の力 | 東西風・気圧勾配 | 権力の横方向の流れ |
| yY | 縦方向の力 | 南北温度勾配 | 上下の権威構造 |
| zZ | 鉛直方向の力 | 上昇気流 | 制度的チェック機能 |
| Uu | 時間変化率 | 気象の傾向 | 変化の速度・勢い |
| vV | 流体・意思 | 水蒸気フラックス | 国民意思・市場心理 |
| wW | エネルギー | 熱移流 | 正統性・動機のエネルギー |

第7項以降（hH前歴、sS規模、eE境界条件、nN非線形臨界…）は、予想と現実の乖離を検証することで発見されていく。

### E式（動的記述）
```
E = (xX + yY + zZ) ⊗ (ωx, ωy, ωz) + rR + iI + jJ
```
- **⊗ テンソル積**：空間3成分が角速度ベクトルと結合。回転・渦・変化の方向を記述。
- **rR**：実部・観測可能な顕在量
- **iI**：虚部・観測できない潜在量（「見えない裏面」の数学的表現）
- **jJ**：交差項・複数の力の相互作用

> **iIがrRに転換する瞬間を捉えることが、この理論の実践的な核心である。**

## 検証の方法

予想と現実を三列で並べる。

| 観測地点 | TTT理論予報 | データ予報 | 実際の現象 |
|---------|-----------|---------|---------|
| 地点A | 支配変数から予想 | 既存モデルの出力 | 観測された事実 |

**乖離の分類：**
1. **量的乖離**：大きさが違う → どの変数の係数が誤っているか
2. **位置的乖離**：場所が違う → 空間解像度の問題か
3. **時間的乖離**：タイミングが違う → Uu（変化率）の読み誤りか
4. **質的乖離**：現象の種類が違う → 欠落している変数の存在を示す

乖離の中に、次の変数（第7項以降）が眠っている。

## 風を読む実践的な問い

あなたの領域で、次の問いを立ててみてほしい。

1. **表面として観測できているものは何か？**
2. **その表面を押した時、どの方向に・どの程度・裏面が変形するか？**
3. **見えていない変数（iI）はどこに潜んでいるか？**
4. **予想と現実が乖離した時、欠落していた変数は何か？**
